Introducción

La estadística y la probabilidad, y en particular la estimación de las funciones de densidad de probabilidad, son fundamentales en el procesamiento de señales e imágenes, ya que permiten modelar la incertidumbre, evaluar la calidad de las predicciones y tomar decisiones informadas.

Objetivo

Generar al estudiante la competencia de creación de código, utilizando formulaciones matemáticas como guía.

Preguntas orientadoras

  1. Encuentre un dataset de señales de origen fisiológico. Para lo anterior puede utilizar portales tales como kaggle o PhysioNet. Una vez se elija el conjunto de datos publique en el espacio de teams Espacio de publicación laboratorio 01, el nombre y la url del conjunto de datos. Dos grupos no pueden tener el mismo dataset, en caso de existir una coincidencia entre dos o más equipos, el primer grupo que publique dentro del espacio, ganará el derecho de utilizar ese dataset, el resto deberán cambiarlo.

  2. Con el conjunto de datos seleccionado, identifique la siguiente información:

    1. Como se origina la señal?
    2. Número de muestras.
    3. Tasa de muestreo.
    4. Espacio Muestral
    5. Elija dos señales aleatorias del conjunto de datos. Las señales se llamarán data01 y data02.

  3. Utilizando el siguiente código:

    counteos, grupos  = np.histogram(data01, bins=gr, density=den)
print(conteos)
print(grupos)

    Responda cual es la salida del código para los siguientes casos

    1. gr=10, den = False
    2. gr=100, den = False
    3. gr=10, den = True
    4. gr=100, den = True
    5. Concluya sobre la naturaleza del código. ¿Que está sucediendo en cada ejecución del código?

  4. Genere un flujo de trabajo para comparar las dos señales. Sugerencia: utilice correlación y matrices de covarianza.

  5. Utilizando el artículo “A COMPARATIVE STUDY OF VARIOUS PROBABILITY DENSITY ESTIMATION METHODS FOR DATA ANALYSIS” de Assenza et al. Generar una función que permita la estimación de la función de densidad de probabilidad utilizando alguno de los métodos presentado en el artículo.
