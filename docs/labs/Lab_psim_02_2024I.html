Introducción

El procesamiento de señales biomédicas es crucial para extraer información relevante de órganos o sistemas biomédicos, contribuyendo al diagnóstico de diversas condiciones. Investigadores en este campo aplican técnicas lineales y no lineales para caracterizar señales de origen cardiaco,

Objetivo

Generar al estudiante la competencia de creación de aplicaciones a partir del procesamiento de señales y electrocardiografía

Dataset

A large scale 12-lead electrocardiogram database for arrhythmia study

  1. Busque información detallada de la base de datos y explique qué tipo de registros contiene. Utilice el articulo asociado a la base de datos.

  2. Investigue qué es la Variabilidad de Frecuencia Cardíaca (Heart Rate Variability HRV) y cómo calcularla.

  3. Implemente un algoritmo de detección de complejos QRS, que le permita hacer análisis de HRV.

  4. Para por lo menos 5 sujetos de la base de datos de cada grupo etario (5 jóvenes y 5 adultos mayores), realice un análisis de HRV comparando los resultados de manera cuantitativa. Utilice las métricas correspondientes.

  5. Existe relación entre la edad y la variabilidad de frecuencia cardíaca?.
